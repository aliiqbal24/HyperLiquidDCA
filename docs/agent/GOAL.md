# HypeDCA Goal

HypeDCA is an independent recurring trade automation tool for Hyperliquid.

The target user is a general trader, not a developer. The product should feel like a small, trustworthy trading control surface: set the recurring trade, see what happens next, and understand exactly why a run executed, skipped, paused, or failed.

## Product Promise

- Free users can create browser-based recurring schedules.
- Paid users can create cloud schedules that execute 24/7 without requiring their browser or computer to be online.
- HypeDCA never asks for a master wallet seed phrase or master private key.
- HypeDCA uses Hyperliquid API/agent wallet credentials only.

## Non-Goals

- Do not build a generic CRUD dashboard.
- Do not build a full trading terminal.
- Do not imply Hyperliquid affiliation or endorsement.
- Do not add native desktop helpers in v1.
- Do not automatically catch up missed browser-mode runs.

## Brand Direction

Use the product name `HypeDCA`.

The interface should be Apple-inspired: clear hierarchy, restraint, high polish, smooth controls, calm motion, and excellent empty/error states.

The palette can be compatible with Hyperliquid’s dark/mint trading feel, but the product must have a distinct logo, layout, and identity.
